# Full Stack Development -- Angular Demo Projects (2025--26 Winter)

This repository contains multiple Angular demo applications developed
for instructional and laboratory purposes.

------------------------------------------------------------------------

## Project List

-   angular-forms-demo1 -- Template & Reactive Forms
-   databinding-demo -- Data Binding
-   demo1_student-app -- Component Structure
-   routing-app -- Angular Routing
-   role-directive-demo -- Structural Directives
-   simple-directive-demo -- Attribute Directives
-   server-demo -- HTTP & Backend Communication
-   feedback-app -- Form + API Integration
-   ecommerce-ticket -- Mini Application Demo

Each folder represents an independent Angular project.

------------------------------------------------------------------------

## How to Use

### 1. Clone Repository

git clone https://github.com/LOKESHKUMARCBE/fullstack.git

### 2. Navigate to Project Folder

cd project-folder-name

Example: cd databinding-demo

### 3. Install Dependencies

npm install

### 4. Run Application

ng serve

Open browser: http://localhost:4200

------------------------------------------------------------------------

## Requirements

-   Node.js (LTS Recommended)
-   Angular CLI
-   VS Code

To install Angular CLI: npm install -g @angular/cli

------------------------------------------------------------------------

Students may: - Clone the repository - Explore individual Angular
